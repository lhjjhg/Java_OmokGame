# ⚪⚫ Omok Game (Java)

> **Java Swing과 소켓(Socket) 통신을 활용한 실시간 멀티플레이어 오목 게임입니다.**
> 데이터베이스 연동을 통해 사용자 관리 및 전적 기록 기능을 제공합니다.

---

## 📌 1. 프로젝트 개요
- **개발 기간**: 2024.09 ~ 2024.12
- **주요 목적**: 자바 프로그래밍의 네트워크 프로그래밍(Socket)과 데이터베이스(MySQL) 연동 및 GUI(Swing) 구현 능력 함양
- **핵심 특징**: 실시간 1:1 채팅 및 대전, 관전 모드, 전적 관리 시스템

## ✨ 3. Key Functions (주요 기능)

### 👤 유저 시스템 (Login & Sign-up)
- **회원가입 및 로그인**: MySQL 연동을 통한 사용자 계정 관리
- **아이디/비밀번호 찾기**: 등록된 사용자 정보를 기반으로 계정 찾기 기능 지원

### 🎮 게임 플레이 (Battle)
- **실시간 대전**: 서버-클라이언트 소켓 통신을 통한 실시간 돌 착수 및 동기화
- **승리 판정 알고리즘**: 가로, 세로, 대각선 8방향 탐색을 통한 5목 완성 체크
- **착수 표시**: 가장 최근에 둔 돌에 빨간색 표시를 주어 가독성 향상
- **대전 모드 전환**: 대기 유저는 '관전 상태'로 진입하며, 플레이어 이탈 시 자동으로 플레이어로 승격

### 📊 데이터 및 관리
- **전적 시스템**: 승리 시 DB의 승리(Win) 횟수 증가, 패배 및 항복 시 패배(Loss) 횟수 업데이트
- **채팅 기능**: 로비 및 게임 방 안에서 유저 간 실시간 채팅 지원
- **항복 및 초기화**: 항복 버튼 클릭 시 상대방 승리 처리 및 게임판 초기화

## 🧠 5. 핵심 구현 로직
- **다중 접속 처리**: `ServerSocket`과 `Thread`를 활용하여 여러 클라이언트의 동시 접속 및 메시지 송수신 처리
- **동기화 로직**: 서버에서 착수 좌표를 수신받아 모든 클라이언트의 `Board` 객체 상태를 동일하게 업데이트
- **예외 처리**: 차례가 아닐 때 돌을 두거나, 관전자가 착수하려 할 경우 경고 메시지 출력
